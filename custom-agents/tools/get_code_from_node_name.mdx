## Usage

The `get_code_from_node_name` tool is used to retrieve code snippets associated with a specific node name in the knowledge graph. This is useful when you know the exact name of the node you're looking for.

## Parameters

- `node_name` (string): The exact name of the node you want to retrieve code from.
