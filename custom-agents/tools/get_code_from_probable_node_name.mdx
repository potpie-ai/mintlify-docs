## Usage

The `get_code_from_probable_node_name` tool is used to fetch code snippets from the knowledge graph based on a probable node name. This is useful when you have an approximate name of a node and want to retrieve its associated code.

## Parameters

- `probable_node_name` (string): The approximate or probable name of the node you're looking for.
