## Usage

The `get_code_graph_from_node_id` tool retrieves the code graph structure associated with a specific node ID. This tool is particularly effective for Python, JavaScript, and TypeScript code, providing insights into the relationships and structure of the code.

## Parameters

- `node_id` (string): The unique identifier of the node in the knowledge graph.
