## Usage

The `get_nodes_from_tags` tool is used to find nodes in the knowledge graph that are tagged with specific labels. This is helpful for categorizing and retrieving related pieces of information.

## Parameters

- `tags` (list of strings): A list of tags to search for in the knowledge graph.
