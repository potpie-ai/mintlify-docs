# Introduction to Potpie Agents

Potpie offers a suite of specialized AI agents designed to assist developers with various aspects of software development. These agents leverage a knowledge graph and advanced code analysis tools to provide intelligent assistance tailored to your codebase.

## Available Agents

Potpie currently offers five specialized agents:

1. [Codebase Q&A Agent](/agents/qna-agent)
2. [Unit Test Agent](/agents/unit-test-agent)
3. [Integration Test Agent](/agents/integration-test-agent)
4. [Debugging with Knowledge Graph Agent](/agents/debugging-agent)
5. [Code Changes Agent](/agents/code-changes-agent)

Each agent is designed to excel in specific tasks, from answering questions about your codebase to generating test plans and assisting with debugging. In the following sections, we'll explore each agent in detail, including their capabilities and how to use them effectively.
